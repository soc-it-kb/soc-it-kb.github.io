IT-EP-WI-Win-02: Disk Encryption & Secure Boot Configuration

Protect data at rest and prevent offline attacks through mandatory disk encryption and Secure Boot enforcement.

💻 Endpoint hardening Windows baseline IT operations

IT procedure
·
Active
Prerequisites
TPM enabled in BIOS/UEFI Supported Windows version Administrative access
Step-by-step instructions
1
Verify TPM status and confirm Secure Boot is enabled
2
Enable BitLocker on the system drive
3
Configure TPM-based key protection
4
Escrow recovery key to the approved storage location
5
Confirm encryption status is fully complete
Validation & verification
BitLocker status shows Fully Encrypted
Recovery key is retrievable by authorised administrators
Secure Boot confirmed enabled
Rollback & troubleshooting
Suspend BitLocker temporarily for firmware updates only
Re-enable immediately after maintenance is complete
Common errors & tips
Do not store recovery keys locally
Always verify encryption completion before closing the task
