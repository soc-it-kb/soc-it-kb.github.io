Purpose & how to use

This knowledge base replaces static documents with structured, searchable, and continuously maintained content. Use it to execute triage, manage incidents end-to-end, and access governance artefacts — all aligned to SOC tiers and the incident lifecycle.

  • Execute alert triage and response using Runbooks
  • Manage end-to-end incidents using Playbooks
  • Follow standardised processes defined in SOPs
  • Perform tool-specific actions using Work Instructions
  • Access tools, references, and governance artefacts
