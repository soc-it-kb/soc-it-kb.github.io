Cyber Security Knowledge Base

The authoritative operational reference for cyber security and IT operations across this institution and its agencies. Structured to support every skill level — from Tier-1 checklists to Tier-3 incident command and GRC oversight — and aligned to NIST CSF 2.0, ISO 27001:2022, CIS Benchmarks, and Zero Trust principles.

Purpose & how to use

This knowledge base replaces static documents with structured, searchable, and continuously maintained content. Use it to execute triage, manage incidents end-to-end, harden the estate, manage attack surface, and access governance and risk artefacts — all aligned to SOC tiers, NIST CSF 2.0, ISO 27001:2022, and Zero Trust principles.

  • Execute alert triage and response using Runbooks
  • Manage end-to-end incidents using Playbooks
  • Follow standardised processes defined in SOPs across SOC, IT Operations, ASM, and GRC
  • Perform tool-specific actions using Work Instructions
  • Manage attack surface through the ASM section — security baselines, vulnerability management, patching, threat intelligence, and network device baselines
  • Govern risk and compliance through the GRC section — risk register, incident response plan, asset management, policies, and forms
  • Access framework references, MITRE ATT&CK mappings, and NIST CSF 2.0 guidance under Tools & References
