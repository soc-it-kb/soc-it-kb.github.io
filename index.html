Purpose & how to use

This knowledge base replaces static documents with structured, searchable, and continuously maintained content. Use it to execute triage, manage incidents end-to-end, harden and manage your attack surface, and access governance artefacts — all aligned to SOC tiers, NIST CSF 2.0, and Zero Trust principles.

  • Execute alert triage and response using Runbooks
  • Manage end-to-end incidents using Playbooks
  • Follow standardised processes defined in SOPs
  • Perform tool-specific actions using Work Instructions
  • Manage attack surface through the ASM section — baselines, vulnerability management, patching, and threat intel
  • Access framework references, MITRE ATT&CK mappings, and NIST CSF 2.0 guidance
