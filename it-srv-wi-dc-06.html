IT-SRV-WI-DC-06: Backup, Recovery & Integrity Validation

Ensure Domain Controllers can be securely and reliably restored following compromise or failure.

🖥️ Server hardening Domain Controller IT operations
🔧 WI ID: IT-SRV-WI-DC-06
·
👤 Author: Paulo Ferreira
·
📅 Published: 13 January 2026
·
IT procedure
Prerequisites
System State backup solution available
Step-by-step instructions
1
Verify System State backups are enabled and scheduled
2
Confirm backups are protected from Domain Admin access
3
Perform periodic restore tests
4
Document offline recovery procedures
Validation & verification
Recent backup confirmed available
Restore test documented with date and outcome
Rollback & troubleshooting
Escalate backup failures immediately to IT Operations and Cyber Security
Common errors & tips
Untested backups provide false assurance — test restore procedures regularly
Backups must be protected from Domain Admin access to survive a full domain compromise
