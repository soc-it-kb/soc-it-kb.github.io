CIS Benchmark Alignment Matrix

Single authoritative reference mapping CIS Benchmark Level 1 controls to the approved Security Baselines — SOPs, Work Instructions, and Validation Checklists — across all covered asset types. Designed for IT teams, Cyber Security, and audit and assurance functions.

CIS Benchmarks Level 1 Implementation-agnostic Audit & assurance reference Published 16 January 2026

This matrix gives IT and Cyber teams a clear, shared reference showing what is mandatory versus recommended across each asset type. It demonstrates explicit CIS alignment for audits and assurance, and prevents control drift or misinterpretation.

This matrix is implementation-agnostic — use it alongside the relevant SOPs and Work Instructions for detailed implementation guidance.
